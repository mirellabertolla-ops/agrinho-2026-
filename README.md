Agrinho 2026: Reflorestação do meio ambiente 

1 - Descrição do projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026, na categoria Programação, subcategoria Ensino Médio. O site aborda o tema “Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”, apresentando reflexões e informações sobre a reflorestação do meio ambiente As árvores protegem o solo da erosão e facilitam a infiltração da água da chuva, ajudando a revitalizar nascentes e rios.

2 - Objetivo

O objetivo deste projeto é a reflorestação de arvores As árvores absorvem dióxido de carbono  um dos principais gases de efeito de estufa.
Proteção hídrica: Raízes fixam o solo e ajudam a recarregar aquíferos, o que faz rios e nascentes voltarem a fluir.


Público-alvo

O público-alvo do reflorestamento divide-se principalmente entre produtores rurais que precisam adequar propriedades ao Código Florestal  governos com metas de restauração de milhões de hectares e empresas que buscam neutralizar emissões de carbono e cumprir agendas ESG

3 - Estrutura do projeto

O projeto é composto pelos seguintes arquivos e pastas:

- `index.html`: arquivo principal com a estrutura da página;
- `style.css`: arquivo responsável pela aparência e organização visual do site;
- `script.js`: arquivo responsável pelas interações e funcionalidades em JavaScript;
- `README.md`: documentação do projeto;
- `assets` ou `img`: pasta utilizada para armazenar imagens e outros recursos.

4 - Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- GitHub
- GitHub Pages ou Vercel 
Toda a lógica e os comportamentos dos elementos são escritos em JavaScript. O p5.js atua como uma camada de abstração, fornecendo funções fáceis de chamar como circle(), rect(), ou loadSound())para controlar coisas complexa

5 - Funcionalidades

O site apresenta as seguintes funcionalidades:

- Organização de conteúdos sobre o tema Agrinho 2026;
- reflorestamento consiste em repovoar áreas degradadas ou desmatadas com espécies vegetais. Essencial para o meio ambiente, essa prática combate as mudanças climáticas ao sequestrar carbono da atmosfera, além de proteger bacias hidrográficas, recuperar o solo e restaurar a biodiversidade. [1, 2, 3, 4, 5]
Os métodos de restauração variam de acordo com o objetivo da área:
Reflorestamento Ecológico: Focado em restaurar a biodiversidade e os serviços ecossistêmicos originais utilizando espécies nativas.
Florestamento Comercial: Voltado para o plantio de árvores de crescimento rápido para atender à demanda por madeira e celulose, reduzindo a pressão sobre florestas nativas. [1, 2, 3]
